Type Definitions

progressListener(state, currentopt, totalopt, erroropt)

Source:
  • jigg.js, line 11

This callback logs or displays progress.

Parameters:
Name Type Attributes Description
state string

one of of the following states: 'connected', 'garbling', 'OT', 'evaluating', 'output', 'error'. OT is called after oblivious transfer is executed for the current party, if state is garbling or evaluating, then current and total are provided.
current number <optional>

Progress so far (i.e., numerator).
total number <optional>

Target total (i.e., the denominator).
error string | Error <optional>

If any error occured, this will be passed with state 'error'.
