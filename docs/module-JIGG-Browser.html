Module: JIGG-Browser

Main client module: garbled circuit protocol agents. This is the module exported by dist/jigg.js for browsers. When included using a script tag, this module exposes the global constructor JIGG, which is an alias for the class @see {Agent}.

Type Definitions

progressListener(state, currentopt, totalopt, erroropt)

This callback logs or displays progress.
Parameters:
Name Type Attributes Description
state string one of of the following states: 'connected', 'garbling', 'OT', 'evaluating', 'output', 'error'. OT is called after oblivious transfer is executed for the current party, if state is garbling or evaluating, then current and total are provided.
current number <optional>
 Progress so far (i.e., numerator).
total number <optional>
 Target total (i.e., the denominator).
error string | Error <optional>
 If any error occured, this will be passed with state 'error'.
